# Memory and Pulse Operations

Mobius OS uses Mobius Memory and Mobius Pulse to power organizational workspaces.

## Memory Use

Mobius OS displays verified organizational history, decisions, workflows, voice summaries, and repository activity from Mobius Memory.

## Pulse Use

Mobius OS displays organizational metrics, department health, workflow efficiency, and executive KPIs from Mobius Pulse.

## Command Center

Command Center is the executive interface inside Mobius OS and should consume both services.
